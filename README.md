# How-s-price-predictor-linear-regression-
🏠 House Price Predictor using Linear Regression

## 📌 Project Overview

This project predicts house prices using the California Housing Dataset and a Linear Regression model built with Scikit-learn.

## 🚀 Features

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Linear Regression Model
- Model Evaluation
- Actual vs Predicted Plot
- Residual Analysis

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## 📊 Evaluation Metrics

- MAE
- RMSE
- R² Score
